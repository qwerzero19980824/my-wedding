# 平行宇宙的相遇 — 求婚叙事网站

> 当前版本 **v3.24.0** | 2026-07-14

纯前端沉浸式求婚叙事网页。三图层 z-index 堆叠架构：视频背景 → 透明文字光晕 → Three.js 六边形网纱布料模拟。拖拽掀开 + 粒子特效 → 翻开我们的故事 → 双轨视差 → 相遇 → 合并记忆 → 故事模块 → 28 张拍立得照片墙 → 单页路线地图 → 最终求婚。支持参数保存、内容编辑、故事模块、每页自由添加文本框/照片框，新增框会作为页面内容层的一部分随页面滚动和淡入淡出，并会在下次打开时自动读取。

## 三图层架构 (v2.0.0 核心)

```
 ┌──────────────────────────────┐
 │  Layer 3 (z:3)  WebGL Cloth  │  ← 正六边形网纱 · Verlet物理 · 拖拽掀开
 │  ┌────────────────────────┐  │
 │  │ Layer 2 (z:2) DOM Hero │  │  ← 毛玻璃模糊 · 情话编辑 · Say Yes
 │  │  ┌──────────────────┐  │  │
 │  │  │ Layer 1 (z:1)    │  │  │  ← 视频背景 · 边缘模糊遮罩
 │  │  │ Video Background │  │  │
 │  │  └──────────────────┘  │  │
 │  └────────────────────────┘  │
 └──────────────────────────────┘

   Say Yes 点击后 ↓
 ┌──────────────────────────────┐
 │  Dual Track → Meeting → Merged│  ← GSAP ScrollTrigger 滚动叙事
 └──────────────────────────────┘
```

### 技术要点
- **Layer 3 — 六边形网纱布料**：Three.js r160 + 纯手写 Verlet 积分物理引擎
  - 质点-弹簧模型：结构弹簧 + 剪切弹簧 + 抗弯曲弹簧
  - 双通道渲染（FrontSide + BackSide）防半透明穿模
  - 程序化正六边形 alphaMap 纹理（Canvas API 动态生成）
  - Raycaster 高斯衰减权重拖拽，任意位置抓取
- **Layer 2 — DOM Hero Card**：`backdrop-filter: blur()` 毛玻璃效果，contenteditable 情话编辑
- **Layer 1 — 视频背景**：`video/1.mp4` 全屏播放，支持边缘模糊遮罩

## 编辑与保存

右上角 `✏️` 会先要求输入 R2 管理口令并由 Worker 验证；通过后，本标签页进入编辑模式，可以：

- 直接修改页面文字，并拖动小圆点调整文字位置
- 点击双轨故事和共同记忆中的照片占位框，直接上传或更换 5 张核心照片
- 用工具条新增故事、照片、时间线、誓言模块
- “加照片”模块创建后，直接点击模块里的照片区域上传或更换图片
- 单张故事模块照片会压缩到最长边 720px、约 34KB；海报区的“批量上传海报照片”则把 4K 原图保存到本机 IndexedDB、轮换只读取缩略图，可一次选择 80 张以上
- 本机海报图片库不会进入 JSON 内容包，也不会同步到 GitHub；如需手机、其他电脑和访客共享同一批原图，按 `PHOTO_STORAGE.md` 接入 R2/S3 对象存储
- 在当前页面新增自由文本框或照片框，并拖动到需要的位置
- 自由照片框创建后工具栏会自动收起；点击照片区域上传图片，点击“展开编辑工具”可继续添加内容
- 点击自由框控制点里的 `色` 按钮切换背景色；默认透明，可以显示页面背景
- 点击 `撤销` 回退上一步结构/位置/内容变更
- 点击 `保存` 手动保存，下次打开自动读取
- 点击 `现场预演`，保存并退出编辑后快速检查四个关键现场起点；需要时点击“进入全屏展示”
- 点击 `导出内容包` 备份当前文字、排版、故事、相纸照片和参数；换电脑时用 `导入内容包` 恢复
- 工具栏底部会显示本地内容容量；接近建议上限时会变成黄色或红色提醒

| `wedding_route_label_size_v1` | 地图地名字号 |

## 物理参数调校

实时控制面板（左上角），改滑块即时生效：

### 面料参数
| 参数 | 默认值 | 范围 | 效果 |
|------|--------|------|------|
| `hexFill` | 80 | 0–180 | 网眼填充白度 (0=极透, 180=厚白) |
| `hexLineWidth` | 5.0 | 2.0–10.0 | 六边形框线粗细 |
| `opacity` | 0.86 | 0.3–1.0 | 白纱整体不透明度 |
| `blur` | 0.6px | 0–2.5px | 薄纱柔焦微朦胧 |
| `tiling` | 450 | 10–600 | 六边形纹理细腻度 |
| `dragRadius` | 7.0 | 1.0–10.0 | 拖拽连带范围 |
| `stiffness` | 0.35 | 0.1–1 | 面料挺括度 |

### 物理参数
| 参数 | 默认值 | 范围 | 效果 |
|------|--------|------|------|
| `flightForce` | 1.8 | 0.5–4.0 | 掀开后飞行推力 |
| `threshold` | 120 | 50–250 | 触发掀开灵敏度 (px) |
| `damping` | 0.98 | 0.90–0.99 | 空气阻尼 (0.9=快停, 0.99=飘摇) |
| `gravity` | 0.014 | 0–0.04 | 悬垂重力 |
| `wind` | 0.015 | 0–0.04 | 微风扰动强度 |

### 视频/图层参数
| 参数 | 默认值 | 范围 | 效果 |
|------|--------|------|------|
| `videoOpacity` | 0.92 | 0.3–1.0 | 视频层透明度 |
| `edgeBlur` | 0px | 0–40px | 边缘模糊遮罩强度 |
| `domBlur` | 7px | 0–30px | 首屏文字层背景柔焦强度 |
